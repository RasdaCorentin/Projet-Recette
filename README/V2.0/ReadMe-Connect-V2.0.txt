Ajouter un salt dans la colonne "setUrlConnect" lors de l'inscription d'un nouvel utilisateur.

Puis lui envoyez un e-mail afin qu'il puisse se connecter.

Rendre les e-mails des utilisateurs unique.

Enlever les potentielles "/" du salt de "setUrlConnect", sinon l'url ne sera pas valide.

Une fois que l'utilisateur à utiliser la page de connect passer la ligne "setUrlConnect" à NULL.
